# Test Code

Jestзҡ„жё¬и©ҰзўјдёҚз”ЁеғҸе…¶д»–зҡ„жё¬и©ҰеҘ—д»¶еңЁ*.test.tsжҲ–*.test.jsйӮ„иҰҒimport, requireе…¶е®ғеҘ—д»¶ з’°еўғиЁӯе®ҡеҘҪ зӣҙжҺҘеҜ«describe, test, beforeAll, beforeEach, afterAll, afterEach, expect, mock, ....... е°ұжҳҜзӣҙжҺҘеҜ«жё¬и©Ұcodeе•ҰпјҢдёҚз”ЁеҶҚimportжңүзҡ„жІ’зҡ„

test codeи·Ҝеҫ‘жҲ‘жҳҜж”ҫеңЁ<rootDir>/test/*.test.ts (йҖҷи·Ҝеҫ‘жҳҜеҸғиҖғеҫҲеӨҡopen sourceзҡ„)

жё¬и©ҰжҳҜеҖӢеҫҲж·ұеҘ§зҡ„й ҳеҹҹ жё¬и©ҰжҖ§иіӘжңүеҲҶеҫҲеӨҡзЁ® е–®е…ғжё¬и©Ұ ж•ҙеҗҲжё¬и©Ұ e2eжё¬и©Ұ... е–®е…ғжё¬и©Ұзҡ„е–®е…ғеӨ§е°ҸжҖҺйәјеҲҮ зӯүзӯүпјҢжҲ‘йӮ„жІ’жңүеҜ«еӨӘеӨҡжё¬и©Ұ дҪҶжңүеҜ«жё¬и©ҰжңғзҷјзҸҫпјҢзЁӢејҸжһ¶ж§Ӣжңғи®Ҡеҫ—еҘҪеҫҲеӨҡ иҰҒжё¬и©Ұзҡ„methodзҡ„side effectжңғж¶ҲеӨұ(жңүside effectжңғзҷјзҸҫи¶…йӣЈжё¬) йқһеёёзҡ„и§ЈиҖҰпјҢд»ҘOOPдҫҶи¬ӣе°ұжҳҜжңғз”ЁеӨ§йҮҸзҡ„DIпјҢзӣёдҫқеӨӘй«ҳд№ҹйқһеёёзҡ„дёҚеҘҪжё¬и©Ұ FPзҡ„и©ұпјҢе°ұжҳҜеҗ„зЁ®curryжҲ–жҳҜclosure зІҫзҘһе°ұжҳҜдёҚиҰҒеңЁmethodиЈЎйқўеҸҰеӨ–е»әз«Ӣе…¶д»–зҡ„зү©д»¶е·Ҙе…·пјҢйҖҷжЁЈжңғйқһеёёзҡ„йӣЈжё¬и©Ұ
